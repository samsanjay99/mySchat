## Getting Started

### Registration
1. Go to http://localhost:5000
2. Click "Get Started"
3. Enter your details:
   - Full Name
   - Email
   - Password
4. Verify email (check for OTP)
5. Login with credentials

### Your Schat ID
After registration, you'll receive a unique Schat ID (e.g., `SCHAT_8KD21A`). Share this with others to start chatting.

1. Click the AI icon in bottom navigation
2. Or click "Ask Super AI" in the search bar

#### Text Conversations
- Ask any question

#### Image Generation
Generate AI images using the `/create` command:

```
/create a beautiful sunset over mountains
/create a cute cat wearing a wizard hat
/create a futuristic city with flying cars
```

**Details:**
- Generation time: 10-15 seconds
- Image size: 1024x1024 pixels
- Click image to view full size
- Rate limit: 10 images/minute
